# 健身房管理系统 java1151

## 项目概述

本项目是一款基于Java与SpringBoot框架开发的健身房管理系统。该系统采用前后端分离的设计模式，前端使用Vue，后端采用Java、SpringBoot、Mybatis等技术栈，数据库使用Mysql5.7，能够为健身房提供全面的业务管理功能。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 运行环境
- 系统环境：Windows/Mac

### 开发环境
- 开发工具：IDEA
- JDK版本：1.8
- 项目构建：Maven
- 数据库：Mysql

### 技术组件
- Java
- SpringBoot
- Mybatis
- Vue
- Mysql5.7

## 功能模块

### 用户功能
- 用户登录

### 管理员功能
- 会员管理
- 健身房管理
  - 课程管理
  - 器材管理
- 员工管理
- 报名表管理

### 会员功能
- 课程查询
- 在线充值
- 充值记录
- Vip管理
- 购买记录
- 我的资料
- 修改密码
